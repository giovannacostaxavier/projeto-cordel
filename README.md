# 📜 Projeto Cordel Moderno – Efeito Parallax

Este projeto é uma página web inspirada no estilo de **cordel**, desenvolvida com **HTML e CSS**, utilizando o efeito **Parallax** para criar uma experiência visual imersiva durante a leitura do poema.

A proposta foi unir literatura e tecnologia, criando uma apresentação moderna para um poema tradicional.

---

## 🌄 Demonstração

🔗 Acesse o projeto online:  
👉 https://giovannacostaxavier.github.io/projeto-cordel/

---

## ✨ Sobre o Projeto

O projeto apresenta um poema no estilo cordel com:

- Efeito **Parallax** nas imagens de fundo  
- Seções alternadas entre texto e imagem  
- Experiência visual dinâmica durante o scroll  
- Layout organizado e harmônico  
- Estilização focada na leitura e estética  

O efeito Parallax cria a sensação de profundidade, tornando a navegação mais envolvente.

---

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3  
- Efeito Parallax com `background-attachment: fixed`  
- Visual Studio Code  

---

## 📂 Estrutura do Projeto

- `index.html` – Estrutura da página  
- `style.css` – Estilização e aplicação do efeito Parallax  
- `cordel-moderno.txt` – Texto base do poema  
- `imagens/` – Imagens utilizadas no projeto  

---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido para:

- Praticar efeitos visuais com CSS  
- Trabalhar com imagens de fundo e profundidade  
- Criar uma experiência diferenciada usando apenas HTML e CSS  
- Explorar criatividade no desenvolvimento front-end

- Scroll e profundidade visual
